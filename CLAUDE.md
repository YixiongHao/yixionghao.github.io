This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

GitHub Pages personal website for Yixiong Hao. Simple static site using HTML/CSS only.

## Site Structure

- **Main page (index.html)** - Intro text, email, links to socials/resume/calendar
- **/writing** - Blog/writing subpages with links
- **/projects** - Portfolio with text, hyperlinks, images/video

## Style Guidelines

- Simplistic, sophisticated, visually appealing, easy to read
- Hyperlinks: different colored font, underline on hover
- Consistent placement/alignment across all pages

## Tech Stack

HTML/CSS only (no frameworks unless justified)

## Implementation Notes

**Colors:**
- Text primary: #1a1a1a
- Text secondary: #5c5c5c
- Link color: #2d5a45 (dark emerald/moss green)
- Background: #f5f0e8 (light beige)

**Typography:**
- Font: Inter (Google Fonts), with system sans-serif fallbacks
- Base size: 16px
- Line height: 1.6
- Max content width: 640px

**File Structure:**
- index.html (main page)
- styles.css (shared styles)
- writing/index.html
- projects/index.html
